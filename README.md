
# Front End Challenge

This is a Landing page devloped for a Front End Challenge


## How To Run

First, run the development server:

```bash
npm install
# or
yarn install

# then you can use

npm run dev
# or
yarn run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Technologies Used

- React
- Next JS
- Styled-Components
- Type Script
- Axios
- Swipper JS
- React Fast Marquee
